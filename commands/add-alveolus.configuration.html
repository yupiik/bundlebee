    Add-alveolus

    Adds a service to the alveoli available in the defined folder. It is a skaffolding command.

    Name: add-alveolus.

    Configuration

    --alveolus (BUNDLEBEE_ADD_ALVEOLUS_ALVEOLUS)

    Alveolus name.

    --image (BUNDLEBEE_ADD_ALVEOLUS_IMAGE)

    Deployment image.

    --manifest (BUNDLEBEE_ADD_ALVEOLUS_MANIFEST)

    Manifest to add the aveolus inside. Default value: ./bundlebee/manifest.json

    --overwrite (BUNDLEBEE_ADD_ALVEOLUS_OVERWRITE)

    If false, generated files already existing will be skipped. Default value: false

    --type (BUNDLEBEE_ADD_ALVEOLUS_TYPE)

    Alveolus/template type. web will create a ConfigMap, Deployment and Service. Default value: web
